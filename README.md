##set up

### install bundler and gems
```
gem install bundler
bundle install --path vendor/bundler
```

### install redis(mac os)

```
brew install redis
```

### setup api keys
you can put your apikeys, but it's easy to rake to set default api key

check some tasks with

```
rake --tasks
```
